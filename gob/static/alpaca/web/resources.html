Resources - Tools and Addons

We've collected several resources that Alpaca users will find interesting and useful as they build and deliver forms for their end users. Some of these resources are works in progress.

Have something you'd like to showcase? Please contact us to let us know.

Alpaca Designer - Drag-and-Drop Visual Builder

We're putting together Alpaca Designer - a drag-and-drop visual builder that helps you compose your Alpaca forms. Modify the schema and options for each field and visually specify the layout and presentation of your form elements.

In addition, Alpaca Designer lets you work on your forms with automatic updates in real-time so you can see how changes to your schema, options or data are reflected right away.

Alpaca Designer generates the JSON and JavaScript source needed to include your form in your web page.

Alpaca Unit Tests

Interested in testing Alpaca to see if it works properly in a specific browser?

We've put our unit tests online along with all of our documentation samples and example code. Give the tests and if anything goes sideways, let us know by submitting an issue to the Alpaca project.
